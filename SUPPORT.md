# Support

## Getting help

| Channel | Use for |
|---------|---------|
| [GitHub Issues](https://github.com/jiaxiantao/3d-express-warehouse/issues/new/choose) | Bugs, feature ideas, integration questions |
| [Live demo](https://jiaxiantao.github.io/3d-express-warehouse/) | Try the app without installing |
| [Architecture](documentation/ARCHITECTURE.md) | How the 3D scene and data layer fit together |

Before opening an issue, search [existing issues](https://github.com/jiaxiantao/3d-express-warehouse/issues) and include your browser, OS, and steps to reproduce when reporting bugs.

## Commercial / custom integration

This project is **MIT-licensed** and free to use in your own products. There is no official paid support tier. For WMS integration or custom layouts, you can:

- Fork the repository and adapt `src/lib/warehouse-data.ts`
- Open a [feature request](.github/ISSUE_TEMPLATE/feature_request.yml) to discuss upstream improvements

## Security issues

Do **not** use public issues for vulnerabilities. See [SECURITY.md](SECURITY.md).
