# Governance

**3D Express Warehouse** is an open-source project maintained by [@jiaxiantao](https://github.com/jiaxiantao) and community contributors.

## Decision making

- **Day-to-day changes** merge via pull request after CI passes and a brief review.
- **Larger changes** (new WMS contracts, breaking layout APIs, major rendering refactors) should start with an issue for alignment.
- **Releases** follow [Semantic Versioning](https://semver.org/) and [Keep a Changelog](https://keepachangelog.com/).

## Roles

| Role | Responsibility |
|------|----------------|
| **Maintainer** | Triage issues/PRs, cut releases, enforce [Code of Conduct](CODE_OF_CONDUCT.md) |
| **Contributor** | Anyone who opens a PR or issue under the project license and CoC |

There is no formal steering committee. The maintainer may add co-maintainers over time based on sustained contributions.

## Conflict resolution

1. Discuss in the relevant issue or PR thread.
2. If behavior violates the [Code of Conduct](CODE_OF_CONDUCT.md), report to the maintainer privately.
3. As a last resort, the maintainer may lock threads or decline contributions that harm project health.
